
  # Optiload app

  This is a code bundle for Optiload app. The original project is available at https://www.figma.com/design/uCXzMa0y6dWNuPwGCiLma6/Optiload-app.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
## Backend connection
Set `VITE_API_BASE_URL` (default `http://localhost:8000/api/v1`) to connect frontend to FastAPI backend.
